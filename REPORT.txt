(1) What will you do? (1-2 paragraphs)

We will try to write a working game boy color emulator that can run a Pokemon game. 
We must find a way to create the window to play the emulator, as well as figure out which instructions are necessary 
to run the game. Important features will be using the mmap-ed IO to handle inputs from the keyboard, as well as 
instructions on displaying pixels and playing sound, if we have the time. 

Ruiqi also mentioned something about dynamic binary translation. At the moment, we will need to see what we are capable 
of finishing and what we are not, but if we do finish on time we will look into doing this as well. We will try to make 
the emulator playable and not too slow.
